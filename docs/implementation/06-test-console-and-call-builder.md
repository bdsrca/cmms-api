# 06 - Test Console And Call Builder

The Test Console can call:

- `/api/ai/summarize-work-order`
- `/api/ai/extract-work-order-fields`
- `/api/ai/cmms-intake`

It supports selecting an environment code and viewing structured JSON responses.

The API Call Builder generates:

- PowerShell `Invoke-RestMethod`
- curl
- JSON request body

Generated examples include `environment_code` when selected.
